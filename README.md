# 🚚 EcoRoute Logistic AI

## 📌 Descripción

EcoRoute Logistic AI es un prototipo desarrollado como parte de la asignatura de Ingeniería de Requisitos.

El sistema busca optimizar la gestión logística en la Ruta Austral de Chile mediante herramientas digitales que permiten el seguimiento de despachos, gestión de conductores, cálculo de viáticos y alertas climáticas.

---

## 🎯 Objetivos

- Mejorar la trazabilidad de los despachos.
- Facilitar el seguimiento para clientes.
- Optimizar la asignación de cargas.
- Entregar información en tiempo real a conductores y administradores.
- Incrementar la seguridad mediante autenticación de acceso.

---

## ⚙️ Funcionalidades Implementadas

### 📦 Portal Cliente
- Consulta de estado del despacho.
- Visualización de ubicación.
- Información climática.

### 🚛 Aplicación del Conductor
- Registro de paso por aduana.
- Confirmación de entrega.
- Recepción de alertas climáticas.

### ⚙️ Dashboard Administrativo
- Asignación automática de cargas.
- Visualización de estado de despacho.
- Seguimiento GPS simulado.
- Gestión de alertas meteorológicas.
- Cálculo de viáticos.

### 🔐 Seguridad
- Acceso protegido mediante autenticación 2FA.
